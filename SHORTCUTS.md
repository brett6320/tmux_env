# Shortcuts

## Prefix and basics

- Prefix: **Ctrl-a** (then another key)  
- Send literal prefix to app: `Ctrl-a` then `Ctrl-a`  
- Reload config: `Ctrl-a` then `r`  
- Detach session: `Ctrl-a` then `D`  

## Windows and panes

- New window in current dir: `Ctrl-a` then `c`  
- Kill current window: `Ctrl-a` then `&`  
- Kill current pane: `Ctrl-a` then `x`  
- Toggle last active pane: `Ctrl-a` then `Space`  
- Zoom/unzoom current pane: `Ctrl-a` then `z`  
- Vertical split (left/right): `Ctrl-a` then `|`  
- Horizontal split (top/bottom): `Ctrl-a` then `-`  
- Move focus left: `Ctrl-a` then `h`  
- Move focus down: `Ctrl-a` then `j`  
- Move focus up: `Ctrl-a` then `k`  
- Move focus right: `Ctrl-a` then `l`  
- Resize pane 5 cells left: `Ctrl-a` then `H`  
- Resize pane 5 cells down: `Ctrl-a` then `J`  
- Resize pane 5 cells up: `Ctrl-a` then `K`  
- Resize pane 5 cells right: `Ctrl-a` then `L`  
- Swap pane with previous: `Ctrl-a` then `<`  
- Swap pane with next: `Ctrl-a` then `>`  
- Toggle synchronize-panes: `Ctrl-a` then `S`  

## Copy mode and clipboard

- Enter copy mode: `Ctrl-a` then `[`  
- Inside copy mode (vim keys enabled):  
  - Start selection: `v`  
  - Toggle rectangle selection: `r`  
  - Copy selection and exit: `y`  
- On Linux with `xclip` installed: `y` copies to system clipboard.[4]
- On macOS with `pbcopy` installed: `y` copies to system clipboard.[4]

## No-prefix navigation and QoL

- Move to pane left: `Alt-Left`  
- Move to pane right: `Alt-Right`  
- Move to pane up: `Alt-Up`  
- Move to pane down: `Alt-Down`  
- Clear screen and scrollback: `Ctrl-a` then `Ctrl-l`  

If you want this as a ready-to-print cheat sheet or with defaults (like `Ctrl-a` then `n`/`p` for next/prev window), that can be added on top of this list.
